# Getting started
Clone the repository.
```bash
git clone https://github.com/MichaelGrigoryan25/chattly-app.git
```
## Installing Dependencies
Download and Install [Node.js](https://nodejs.org/en/).
```bash
npm install express
npm install socket.io
npm start
```
